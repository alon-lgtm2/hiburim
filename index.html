חיבורים - קונספטים לעיצוב

10 כיוונים עיצוביים ייחודיים לאתר הקהילה. כל קונספט הוא עמוד בית שלם ומלא - לחצו לצפייה.
כל העיצובים הם רספונסיביים, תומכים בעברית מלאה (RTL), וכוללים אנימציות ואינטראקציות.
01

Warm Community Hub

בית קהילתי חם
עיצוב חם ומזמין שמרגיש כמו בית. שילוב של טורקיז עמוק עם קורל חם יוצר תחושה מקצועית אך אישית. כרטיסי מטפלים עגולים, אקורדיון שאלות ותשובות, ניווט שקוף שמשתנה בגלילה, ומפרידי סקשנים מעוגלים.
גופנים: Assistant + Heebo
סגנון: חם, עגול, קהילתי
כרטיסים עגולים אקורדיון FAQ ניווט שקוף מפרידים מעוגלים

Modern Minimalist

מינימליזם מודרני
אלגנטיות שקטה בהשראת מגזיני עיצוב. הרבה מרחב לבן, טיפוגרפיה סריפית עדינה, גוונים של ירוק מרווה וורוד מאובק. כל אלמנט מדויק ומחושב. תחושת בוטיק יוקרתי שלא צועק אלא לוחש.
גופנים: Frank Ruhl Libre (סריף) + Heebo
סגנון: מינימלי, עריכתי, אלגנטי

Organic Nature

אורגני וטבעי
עיצוב בהשראת הטבע עם צורות אורגניות זורמות, גווני אדמה, עלים ומפרידי גלים. Blob shapes מתחלפים, כרטיסים עם פינות אורגניות, ואיור עץ צומח ב-CSS. תחושה של צמיחה, שורשים, וטיפוח.
גופנים: Secular One + Heebo
סגנון: אורגני, טבעי, שורשי
Blob Shapes מפרידי גלים פינות אורגניות איור CSS

כתב יד ענק סגול וזהב שובר מוסכמות אנימציות דרמטיות

Mediterranean Warmth

חום ים-תיכוני

מה מתחת למכסה המנוע?

במקום להכביר במילים, בחרנו להראות לכם הוכחת התכנות. אבל חשוב שתדעו - מאחורי העיצוב היפה עומדת תשתית טכנולוגית מתקדמת שתלווה אתכם לשנים.

⚙

מערכת ניהול תוכן (CMS) מלאה

ממשק ניהול בעברית שמאפשר לכל אחת מכן לעדכן את הפרופיל שלה, להעלות מאמרים, לערוך תכנים ולהוסיף אירועים - בלי לגעת בקוד. פשוט, אינטואיטיבי, ובשליטה מלאה שלכן.

Headless CMS WYSIWYG Editor Media Upload Draft / Publish
🔒

אבטחה ברמה הגבוהה ביותר

תעודת SSL/TLS (HTTPS) מובנית, הגנה מפני התקפות XSS ו-CSRF, Content Security Policy, ו-Rate Limiting. האתר שלכן מוגן כמו כספת - אנחנו לא מתפשרים על אבטחה.

SSL/TLS CSP Headers GDPR Ready WAF Protection
🚀

ביצועים וזמני טעינה

האתר בנוי עם Static Site Generation (SSG) ו-CDN גלובלי - מה שאומר שהוא נטען בפחות משנייה מכל מקום בעולם. ציון מושלם ב-Google PageSpeed Insights (Core Web Vitals).

SSG / ISR Global CDN Image Optimization Lazy Loading
🔍

SEO מתקדם - שגוגל אוהב

Structured Data (Schema.org) למטפלים, Open Graph tags לשיתוף ברשתות, Sitemap אוטומטי, ו-Semantic HTML. האתר שלכן יהיה גלוי למי שמחפש מטפל/ת בעברית בגוגל.

Schema.org Open Graph Auto Sitemap Meta Tags
📱

רספונסיבי לכל מסך

Mobile-First Design - האתר נראה מושלם בנייד, טאבלט ומחשב. כל אלמנט מותאם, כל טקסט קריא, וכל כפתור נגיש. יותר מ-70% מהגולשים מגיעים מהנייד - אנחנו מוכנים.

Mobile First Fluid Typography Touch Optimized Breakpoints
🇧🇮

עברית מלאה - RTL Native

לא תרגום, לא hack - עברית מהיסוד. כיווניות RTL מובנית בכל רכיב, גופנים עבריים מותאמים (Heebo, Assistant, Frank Ruhl Libre), וטיפוגרפיה שמכבדת את השפה.

RTL Native Hebrew Fonts Bidi Support i18n Ready
💻

הקוד שלכן - לגמרי

בעלות מלאה על הקוד (Open Source). אין נעילה לספק, אין מנוי חודשי לפלטפורמה. תרצו להחליף מפתח מחר? הקוד נקי, מתועד, ומוכן להעברה. זה שלכן, נקודה.

Git Repository Clean Code No Lock-in Documentation
⚡

גמישות ויכולת הרחבה

ארכיטקטורה מודולרית שצומחת איתכן. היום אינדקס מטפלים, מחר קבוצות אונליין, סדנאות, או בלוג. Component-Based Architecture שמאפשרת להוסיף פיצ'רים בלי לשבור דברים.

Component Based API Ready Modular Scalable
